# Dashboard - Панель управления

Отдельное React.js приложение для администраторов и модераторов платформы коммерческой недвижимости.

## Технологии

- **React 18** - UI библиотека
- **Vite** - Сборщик и dev сервер
- **React Router v6** - Маршрутизация
- **Tailwind CSS** - Стилизация
- **Axios** - HTTP клиент
- **Lucide React** - Иконки

## Установка

```bash
# Установить зависимости
npm install

# Запустить dev сервер
npm run dev

# Собрать для продакшена
npm run build

# Предпросмотр продакшен сборки
npm run preview
```

## Доступ

Dashboard доступен только для пользователей с ролями:
- **admin** - полный доступ ко всем разделам
- **moderator** - доступ к модерации объектов и аналитике

## Разделы

- **Обзор** - Статистика и быстрые действия
- **Объекты** - Управление и модерация объявлений
- **Пользователи** - Управление пользователями (только admin)
- **Аналитика** - Детальная статистика
- **Платежи** - История транзакций (только admin)
- **Отзывы** - Управление отзывами
- **Отчеты** - Генерация отчетов (только admin)
- **Настройки** - Конфигурация системы (только admin)

## Переменные окружения

Создайте файл `.env` в корне проекта:

```env
VITE_API_URL=http://localhost:3001
```

## Dev сервер

Dashboard запускается на порту **3002** (настроено в `vite.config.js`)

```bash
npm run dev
# Откройте http://localhost:3002
```

## Продакшен

```bash
npm run build
# Файлы будут в папке dist/
```
