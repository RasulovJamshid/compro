# Добавление тестовых данных

## Вариант 1: Через Docker (Рекомендуется)

```bash
# Скопируйте SQL файл в контейнер и выполните
docker cp backend/src/database/seeds/simple-seed.sql realestate-db:/tmp/seed.sql
docker exec -it realestate-db psql -U postgres -d realestate -f /tmp/seed.sql
```

## Вариант 2: Локально через psql

```bash
psql -h localhost -U postgres -d realestate -f backend/src/database/seeds/simple-seed.sql
```

## Вариант 3: Через pgAdmin или DBeaver

1. Откройте файл `backend/src/database/seeds/simple-seed.sql`
2. Скопируйте содержимое
3. Вставьте в SQL редактор вашего клиента
4. Выполните запрос

После выполнения скрипта проверьте:

